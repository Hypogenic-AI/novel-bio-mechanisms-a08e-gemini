# Cloned Repositories

## InterPLM
- **URL**: https://github.com/ElanaPearl/interPLM
- **Purpose**: Toolkit for discovering interpretable features in PLMs via Sparse Autoencoders.
- **Location**: code/interPLM/
- **Key Features**: Pre-trained SAEs for ESM-2, analysis tools, and visualization dashboard.

## InterProt
- **URL**: https://github.com/etowahadams/interprot
- **Purpose**: Training, evaluating, and interpreting SAEs on PLMs.
- **Location**: code/interprot/
- **Key Features**: Similar to InterPLM, focuses on mechanistic interpretability for biology.
